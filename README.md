# Employee Task Management System

A full-stack web application built using Spring Boot, React.js, PostgreSQL, JWT Authentication, Railway, and Vercel.

## Project Overview

The Employee Task Management System is a full-stack application developed to understand and implement modern web application development concepts. The project allows employees to register, log in securely, and manage their daily tasks through a user-friendly interface.

The main objective of developing this project was to gain hands-on experience with frontend development, backend development, database management, authentication, API integration, and cloud deployment. Through this project, I learned how different technologies work together to create a complete production-ready application.

## Why I Developed This Project

I built this project to strengthen my Full Stack Development skills and gain practical experience in:

* Building REST APIs using Spring Boot
* Developing responsive user interfaces with React.js
* Implementing JWT-based Authentication and Authorization
* Integrating frontend and backend applications
* Managing data using PostgreSQL
* Deploying applications using Railway and Vercel
* Handling CORS, API communication, and production environments

## Deployment Process

### Backend Deployment

* Developed the backend using Spring Boot.
* Configured PostgreSQL database on Railway.
* Connected the application using JDBC.
* Deployed the backend service on Railway.

### Frontend Deployment

* Developed the frontend using React.js.
* Connected React with the Spring Boot APIs using Axios.
* Deployed the frontend application on Vercel.
* Configured production API URLs for backend communication.
